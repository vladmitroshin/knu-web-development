Лабораторная работа №7

Работа с jQuery

Создать HTML страницу с произвольным оформлением и содержимым, где в качестве заголовка указать тему и номер практической работы. Каждое задание поместить на страницу.

  • 1) На странице с помощью jQuery реализовать выполнение следующих действий: по двойному клику на элемент страницы выводить модальное окно со значением атрибута этого элемента.
  • 2) На странице, которая была создана в п.1, реализовать анимацию по следующему алгоритму: Текст синего цвета двигается слева направо до центра страницы, увеличивается, становится желтым, затем перемещается вправо и увеличивается.
  • 3) Добавить на страницу форму, содержащую несколько элементов формы, при этом при выборе определенного значения в одном элементе (см. примеры) отображать другой элемент.
